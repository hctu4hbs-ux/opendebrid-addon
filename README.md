# OpenDebrid

**Author:** Eyad Al-Juhani

## Description

OpenDebrid is a Stremio addon that provides streaming capabilities.

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the server: `npm start`

## Configuration

- **ID:** org.opendebrid.eyad
- **Version:** 1.0.0
- **Supported Types:** Movies, Series
- **Resources:** Stream

## Dependencies

- stremio-addon-sdk: ^1.6.0
- express: ^4.18.2
